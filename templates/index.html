Car Detection Web Application

Project Description

The Car Detection Web Application is a simple yet powerful tool that utilizes computer vision techniques to identify and detect cars in uploaded images. Built with Flask, a lightweight web framework for Python, this application allows users to upload images and receive real-time feedback on the number of cars detected within the image. It leverages the Haar Cascade Classifier, a popular machine learning object detection method, to accurately identify vehicles in various conditions.

Features

  • Image Upload: Users can easily upload images via a web interface.
  • Car Detection: The application processes the uploaded images using OpenCV to detect cars and draws bounding boxes around detected vehicles.
